# Aura Dashboard 🚀
Projeto desenvolvido para consolidar estudos de Front-end e UX.

## ✨ Funcionalidades
- Barra de progresso dinâmica calculada por JS.
- Filtro de módulos por status.
- Modo Escuro com persistência local.
- Frases motivacionais via API.

## 🛠 Tecnologias
HTML5, CSS3, Bootstrap 5, JavaScript Vanilla.
